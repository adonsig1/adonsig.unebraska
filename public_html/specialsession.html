Saturday, October 22, 8:30 - 11:30 am

There are other talks scheduled in the AMS Meeting Program. In particular, there are four Invited Addresses and Sir Michael Atiyah will present a public lecture (tentativley scheduled for Friday, October 21, 4:15 - 5:15 pm).

The AMS imposed deadline for Abstract Submission is August 30, 2005. No talk can appear on the AMS Program if an abstract is not received by this date. The abstract should be sent electronically directly to AMS using the Abstract Submission form.
